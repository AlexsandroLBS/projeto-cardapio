# Instruções para o Front-End

Para rodar o front do projeto, usar versão mais recente do [Node (20.17.0)](https://nodejs.org/pt) 

Para conseguir rodar o front

## Primeiro uso: 
Instalar os pacotes

Ir na pasta cardapio-web
```
npm install
```

## Usos seguintes
Basta ir na pasta cardapio-web
```
npm run dev
```
